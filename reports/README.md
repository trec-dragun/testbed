# Public Reports

This directory contains the public report copies for the May 25, 2026 empirical Codex/OpenRouter leaderboard run shown in the top-level [README](../README.md).

Each model directory contains 30 article folders. Each article folder includes:

- `report.html`: rendered report for readers
- `report.json`: structured report output used for scoring
- `target.txt`: source article text supplied to the model-facing run
